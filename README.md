## File Template

Every topic follows this 10-section structure:

1. What & Why
2. Core Concepts
3. How It Works
4. Code Examples
5. Common Mistakes & Pitfalls
6. When to Use / Not Use
7. Real-World Scenario
8. Interview Questions (with answers)
9. Exercises (with hints)
10. Further Reading
