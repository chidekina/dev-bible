# Track 05 — DevOps

Containerization, CI/CD, web servers, deployment, and observability. Build the infrastructure to ship reliably.

**Estimated time:** 2–3 weeks

---

## Topics

1. [Docker Fundamentals](docker-fundamentals.md) — images, containers, Dockerfile, multi-stage builds
2. [Docker Compose](docker-compose.md) — services, volumes, networks, healthchecks, production patterns
3. [GitHub Actions & CI/CD](github-actions-cicd.md) — workflows, secrets, matrix builds, deploy pipelines
4. [Nginx & Reverse Proxy](nginx-reverse-proxy.md) — config, proxy_pass, SSL, rate limiting, load balancing
5. [VPS Deployment](vps-deployment.md) — server setup, Docker on VPS, Traefik, zero-downtime deploys
6. [VPS Hardening](vps-hardening.md) — SSH, firewall, fail2ban, Docker security, audit tools
7. [Monitoring & Observability](monitoring-observability.md) — metrics, logs, traces, Prometheus, Grafana, Pino

---

## Suggested Order

**1 → 2 → 5 → 6 → 4 → 3 → 7**

> Learn Docker before CI/CD — pipelines build and push Docker images.
