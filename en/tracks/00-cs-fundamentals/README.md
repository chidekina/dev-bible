# Track 00 — CS Fundamentals

Master the foundations of computer science: complexity analysis, data structures, and algorithms. Essential for technical interviews and writing efficient code at scale.

**Estimated time:** 4–6 weeks (deep study)

---

## Topics

1. [Big O Notation](big-o-notation.md) — time/space complexity analysis
2. [Arrays & Strings](arrays-strings.md) — the most fundamental data structure
3. [Linked Lists](linked-lists.md) — singly, doubly, circular
4. [Stacks & Queues](stacks-queues.md) — LIFO/FIFO structures
5. [Hashmaps & Sets](hashmaps-sets.md) — O(1) lookup, collision, hashing
6. [Trees & Binary Search](trees-binary-search.md) — BST, AVL, traversals
7. [Graphs](graphs.md) — BFS, DFS, topological sort, shortest path
8. [Heaps](heaps.md) — min/max heap, priority queues
9. [Tries](tries.md) — prefix trees, autocomplete
10. [Segment Trees & Advanced](segment-trees-advanced.md) — range queries, Fenwick tree
11. [Sorting Algorithms](sorting-algorithms.md) — from bubble sort to radix sort
12. [Searching Algorithms](searching-algorithms.md) — binary search and variants
13. [Recursion & Backtracking](recursion-backtracking.md) — base cases, call stack, pruning
14. [Dynamic Programming](dynamic-programming.md) — memoization, tabulation, classic problems
15. [Greedy Algorithms](greedy-algorithms.md) — greedy choice, interval scheduling
16. [Common Patterns](common-patterns.md) — sliding window, two pointers, fast/slow, merge intervals

---

## Suggested Order

Linear: **1 → 2 → 3 → … → 16** (each topic builds on the previous)

> If you're preparing for interviews, prioritize: Big O → Arrays → Hashmaps → Trees → Graphs → DP → Common Patterns
