# Track 02 — Frontend

Modern frontend development with React and Next.js. Covers component patterns, performance, testing, and web vitals.

**Prerequisite:** Track 01 (especially JavaScript and TypeScript)

**Estimated time:** 3–4 weeks

---

## Topics

1. [React Hooks](react-hooks.md) — useState, useEffect, useRef, useMemo, custom hooks
2. [React Patterns](react-patterns.md) — compound components, HOC, render props, provider
3. [React Performance](react-performance.md) — reconciliation, memoization, code splitting, virtualization
4. [Next.js App Router](nextjs-app-router.md) — Server Components, layouts, Server Actions, route handlers
5. [Next.js SSR, SSG & ISR](nextjs-ssr-ssg.md) — rendering strategies, fetch caching, streaming, edge runtime
6. [Unit Testing](testing-unit.md) — Vitest, Testing Library, mocking, what to test
7. [End-to-End Testing](testing-e2e.md) — Playwright, Page Object Model, CI integration
8. [Core Web Vitals](core-web-vitals.md) — LCP, INP, CLS — measure and fix

---

## Suggested Order

**1 → 2 → 3 → 4 → 5 → 6 → 7 → 8**

> Don't skip testing. Untested React code is technical debt waiting to explode.
